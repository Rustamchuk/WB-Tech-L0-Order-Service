# Stack: Go/Golang, PostgreSQL, Docker Compose, Postman, REST, NATS Streaming

### + Graceful ShutDown

## Запуск проекта:
  `make migrate-up`
  `make run`

## Отскат проекта:
  `Ctrl+C`
  `make migrate-down`
